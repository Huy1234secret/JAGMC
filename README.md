# JAGMC

This repo contains a Minecraft 1.21.1 datapack and matching resource pack for custom JAG tokens.

## Datapack

Folder: `jagmc_tokens_datapack`

Install by zipping the folder and placing it in your world's `datapacks` folder.

Functions (use in-game):

- `/function jagmc:give/bronze`
- `/function jagmc:give/silver`
- `/function jagmc:give/gold`

## Resource pack

Folder: `jagmc_tokens_resource_pack`

Install by zipping the folder and enabling it in Minecraft. This pack adds the custom item models/textures.

### Texture placement (images are not included in the pack)

Place/rename the PNG files exactly like this:

- `JAGMC1.png` → `jagmc_tokens_resource_pack/assets/jagmc/textures/item/bronze_jag_token.png`
- `JAGMC2.png` → `jagmc_tokens_resource_pack/assets/jagmc/textures/item/silver_jag_token.png`
- `JAGMC3.png` → `jagmc_tokens_resource_pack/assets/jagmc/textures/item/gold_jag_token.png`

After adding the textures, zip the `jagmc_tokens_resource_pack` folder and enable it in Minecraft.
